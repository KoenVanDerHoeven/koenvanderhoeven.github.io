Notable Projects

Maximinute

Pennapps 12 Project - HTML/CSS/Javascript/Bootstrap

Web application that takes user input (time) and randomly generates content for the user to waste their time on that will fill the alloted time. Makes use of Youtube API.

Probe

PennPlay 2014 Project - Unity/C#

Unity web game with the theme of a space race. Allows users to select a country and race against each other to be the first to build a rocket. Built using an idea created in a prior PennApps. Won "best use of graphics and art".

Raz-Rover

CIS 191 - Python/Arduino

Build a Arduino buggy used to navigate around a room to parse data to try to find the path with the shortest distance and travel in that direction. Can be controlled using a keyboard and parses data to be used by the user and the robot.
