Notable Projects

Maze sorting algorithm

CIS110 - Java

Solves maze data types using depth first search to traverse through vertices in a maze and recursively explore all possible paths until a path from the start to the end has been found.

Flower Parser

CIS121 - Java

Takes in a database of flowers and upon input of an unknown flower searches throgh the data to predict the species of the flower.

Twitter

CIS121 - Java

Program that parses twitter like data in the form of JSON and using different search algorithms searches for data. The program uses measures of centrality, Dijkstra's algorithm and Kosaraju's algorithm as means of searching for data.

Probe

PennApps 10 Project - Android SDK

Android game with the theme of a space race. Allows users to select a country and race against each other to be the first to build a rocket.

View Project

The Hunter

CIS 120 - Java

A side scrolling platformer game developed that allows the user to select from multiple characters then you have to traverse through levels to win the game.
